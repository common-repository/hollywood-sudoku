A Sudoku game board on any page with a simple shortcode or block in the Gutenberg editor.

== Description ==

The Hollywood Sudoku plugin makes it easy to add an engaging element to any page on your site.  Using the simple
shortcode you can place an interactive, mobile friendly, sudoku game on any page or post.  The game includes 2100 Sudoku puzzles and 
an innovative Hint system to help your players get unstuck on any puzzle.  The Hollywood annimation delights every
user each time they use it.


== Installation ==

1. Install either via the WordPress.org plugin directory, or by uploading the files to your server.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Use the shortcode "sudoku" anywhere you want to place the Sudoku board
1. In Gutenberg editor you will also find the Sudoku block for easy placement on any page

== Frequently Asked Questions ==

= What is Sudoku? =

In classic Sudoku, the objective is to fill a 9×9 grid with digits so that each column, each row, and each of the nine 3×3 subgrids that compose the grid (also called "boxes", "blocks", or "regions") contain all of the digits from 1 to 9.

= Who invented Sudoku? =

 Howard Garns, a 74-year-old retired architect and freelance puzzle constructor from Connersville, Indiana, and first published in 1979 by Dell Magazines as Number Place (the earliest known examples of modern Sudoku)

= How many Sudoku Puzzles Are There? =

Too many to count on two hands.  This plugin includes over two thousand Sudoku boards

= Does this work with WooCommerce? =

Yes, Using a shortcode you can place a Sudoku board anywhere on your site.

= Does this work with Yoast? =

No reason to see why not.  The Sudoku board is not indexable by search engines, but the words you place abour the board certainly are helpful for SEO.

= Does this work with other Popup-Builders and Page-Builders? =

Probably, but if not, please do let us know and we'll work quickly to get your favorite Popup-Builders and Page-Builders supported.  Sudoku was designed primarily for the Gutenberg block editor, but the shortcode should work everywhere else.
